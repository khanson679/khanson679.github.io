Terminal Wizardry

  1. End a text program that is hanging (i.e. in an infinite loop)
    Press ctrl-c.
  2. Remove a folder and all of it's contents.
    Use "rm -rf [folder]". Be extremely careful when using this, since it cannot be undone.
  3. Email text or files
    "mail" is the most basic program for sending email, but it's incapable of sending attachments, so for that you will have to use "mutt" instead. This is the easiest way to send files to me when logged in to Adriatic.
    • Text body only, from a file: 
      mail [address] -s "subject" < text_file
    • Short text body only:
      echo "some text" | mail [address] -s "subject"
    • Short body plus attachment:
      echo "some text" | mutt [address] -s "subject" -a [attachment_file]
  4. Run graphical programs
    Add an '&' at the end of the command to fork the process (run the program in a new process). I you do this, you can press enter to get a prompt and continue using the terminal. If you don't, the prompt won't return until you close the program or enter ctrl-c, and closing the terminal also kills the graphical program.
  5. Enable file/folder colors in "ls"
    Edit your .aliases file (a hidden file in the home directory) and add the following line to the top:        alias ls 'ls --color=auto'
    This makes navigation a bit easier.

Misc.

  1. End a misbehaving graphical program
    Use “ps” to find its “process id” (also printed in terminal when programs started), then “kill <process id>”.s
