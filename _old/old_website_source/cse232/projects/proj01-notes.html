Project 1 Extra Notes

  1. Follow the instructions exactly. If a section gets screwed up, you may want to restart it.
  2. Pay careful attention to the Vim functions that work differently from most text editors:
    1. Copy/paste
    2. Undo
  3. Make sure you do the ".vimrc" section (Lesson 7.2). This will enable various critical features, and make Vim behave much more normally. See Vim Configuration for more info.
