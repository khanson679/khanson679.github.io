LaTeX & Tree Drawing Workshops

Introduction

There are two interrelated goals for these workshops:

  1. To allow you to draw syntactic trees using a computer
  2. To allow you to use LaTeX to write about linguistics

We'll do this in three steps:

  1. Drawing simple non-LaTeX trees
  2. Learn to use the "Qtree" LaTeX package to produce trees using an online LaTeX previewer (no installation required).
  3. Learn to write syntax and semantics papers (or even just homework assignments) using the website ShareLaTeX (still no installation required).

Background

TeX is a kind of programming language used for typesetting documents. LaTeX is essentially a collection of TeX macros that make this significantly easier.

The basic purpose of using LaTeX rather than Microsoft Word or some other desktop publishing program is that it fully separates formatting from content, making it possible to change the formatting for your document by making changes in only one place. And because TeX is a very good typesetter, it's much easier to get professional-looking results.

But the main reason that you really want to learn LaTeX is that it makes writing papers in subjects like math, philosophy, and linguistics much easier. And in the case of syntax, it also allows you to draw perfect trees.

Getting Started

Setting Up an Editor

LaTeX files are created and modified using a text editor. Bracketed trees (even outside of LaTeX) are likewise easiest to produce using an editor.

There are many good editors out there, but for simplicity, everyone should use jEdit, which works on all operating systems (it requires Java). You can download the program here:

http://jedit.org/index.php?page=download

Use the Windows or Mac installer, not the Java installer. Uncheck the boxes for installing the macro API and starting automatically on system startup.
