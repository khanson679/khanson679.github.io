Using a Text Editor

Text editors are primarily designed for programming and other work that involves raw text files, which lack fonts, bold/italics, alignment, and other typographic features. Otherwise, they work much like any other graphical content creation program.

This page assumes you're using jEdit, which works on all operating systems, but most editors will have all of the features discussed on this page. I'm not a Mac user, but on Windows, Notepad++ is free and extremely good.

Super Simple Stuff

Opening and Saving Files

Usually, you'll save your files in your user directory (such as "My Documents" and the like), so no problem there.However, text files can have various extensions, like ".tex" for LaTeX documents, and ".py" for Python programs, so double clicking will not necessarily open the file in your editor of choice.

You can set your file associations to have each extention you use open in jEdit by default. On Windows, you can also right click and choose "Open with jEdit".

Text Manipulation

You'll find the same "file", "edit", and "view" menus as anywhere else, and you can select, cut, copy, and paste text just like usual.

Indentation

In order to format text files using only ASCII characters, we instead rely on intelligent use of whitespace and indentation. Most text editors have features that make this easier.
